Trabajo desarrollado por Santiago Osorio Obando.

El trabajo consta de 2 archivos HTML con el contenido de 2 páginas web, un archivo css para darle estilo a las páginas y una archivo de javascript con las funciones
que permiten que la página tenga interacción con el usuario y no sea totalmente estática.

- Una que permite calcular la suma, resta, multiplicación y división de 2 números.
- Otra que permite visualizar una matriz identidad del tamaño que el usuario indique.

Trabajo para la clase de Técnicas de Programación, docente, Diana Margot Lopez.
